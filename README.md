# Food Calorie Fitness System

This project is a Streamlit-based web application that predicts food categories from images and estimates calorie values.

## Features
- Food image recognition using Deep Learning
- Calorie estimation
- User authentication system
- Admin functionality

## Technologies Used
- Python
- TensorFlow / Keras
- Streamlit
- SQLite
- OpenCV

## Dataset
Food calorie dataset stored in CSV format.
